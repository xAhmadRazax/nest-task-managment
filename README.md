# 📝 Nest Task Management Monorepo

A **monorepo** for a simple **Task Management App** with:

- **Backend:** NestJS REST API with **user authentication**, **task management**, and **task search/filtering**.
- **Frontend:** Minimal UI for creating, updating, viewing, and deleting tasks.
- The frontend **does not implement user profiles or task search/filter UI**, but the backend supports these features.

---

## 🛠️ Features

**Backend:**

- ✅ User authentication (signup/login/change-password/reset-password/forgot-password)
- ✅ CRUD tasks (create, read, update, delete)
- ✅ Subtasks support
- ✅ Task search/filter (backend only)

**Frontend:**

- ✅ Minimal task management UI
- ✅ User authentication (signup/login)
- ❌ No user profile UI
- ❌ No task search/filter UI
